# README проекта

## Инструкции по установке

### 1. Клонирование проекта

Клонировать репозиторий

### 2. Установка зависимостей

- Открыть папку с проектом (или в терминале: cd <путь к файлу>)

- В терминале в корне проекта прописать: 
npm install 

### 3. Запуск проекта

- В терминале в корне проекта прописать:
npx expo start --no-dev --minify

- Скачайте приложение Expo Go на Android или iOS

- Отсканируйте QR-код через Expo Go


## Принятые решения при реализации

### Архитектура состояния

Для управления маркерами используется React Context (MarkerContext),
размещённый в `_layout.tsx`.

Причины выбора: - Централизованное хранение состояния - Доступ к данным
из любого экрана - Отсутствие необходимости в сторонних state-менеджерах

### Навигация

Используется Expo Router с файловой структурой маршрутов: 
- 'index.tsx': экран карты
- 'marker/[id].tsx': экран деталей маркера

### Работа с картой

Используется `react-native-maps`. - Долгое нажатие создаёт маркер -
Нажатие на маркер открывает экран деталей

### Работа с изображениями

Используется 'expo-image-picker': 
- Выбор изображения из галереи 
- Удаление изображений
- Хранение изображений внутри объекта маркера


## Реализованные функции

- Добавление маркера долгим нажатием
- Просмотр деталей маркера
- Добавление изображений
- Удаление изображений
- Удаление маркера с подтверждением



## Известные проблемы и ограничения

1.  Данные не сохраняются после перезапуска приложения
2.  Нет сохранения в локальное хранилище
3.  Нет кластеризации маркеров
4.  Нет редактирования координат маркера
